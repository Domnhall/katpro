Learn MYSQL


What is MySQL?
MySQL is a widely used open-source relational database management system (RDBMS). It helps in managing and organizing data in a structured way using tables.

SQL in full: Structural Query Language

    • MOST Important SQL commands

    • SELECT→ extracts data from a database
    • UPDATE→updates data in a database
    • DELETE→ deletes data from a database
    • INSERT INTO→inserts new data into a database
    • CREATE DATABASE→ creates a new database
    • CREATE INDEX→creates an index (search key)
    • DROP INDEX→deletes an index
    • ALTER TABLE→modifies a table
    • DROP TABLE→ deletes a table
    • ALTER DATABASE→modifies a database
    • CREATE TABLE→creates a new table

    • Basic MYSQL Operations

    • Creating a Database:

    • CREATE DATABASE my_database;

    • Selecting a Database:

    • USE my_database;

    • Creating a Table:

    • CREATE TABLE workers(
      id INT AUTO_INCREMENT PRIMARY KEY,
      name VARCHAR(255),
      position VARCHAR(255),
      hire_date DATE
      );

    • Inserting Data:

    • INSERT INTO workers (name, position, hire_date)
      VALUES ('Joe Bide', 'Software Engineer', '2004-08-13');

    • Querying Data:

    • SELECT * FROM workers;

    • Updating Data:

    • UPDATE workers;
      SET position = 'Expert Software Engineer'
      WHERE name = 'Joe Bide';

    • Deleting Data:

    • DELETE FROM workers;
      WHERE name = 'Joe Bide';

    • User Management

    • Create a User:

    • CREATE USER 'username'@'host' IDENTIFIED BY 'password';

    • Grant Privileges:

    • GRANT ALL PRIVILEGES ON db_name.* TO 'username'@'host';

    • Revoke Privileges

    • REVOKE ALL PRIVILEGES ON db_name.* FROM 'username'@'host';

    • Drop a User

    • DROP USER 'username'@'host';

    • Database Administration

    • Show Databases

    • SHOW DATABASES;

    • Show Tables

    • SHOW TABLES;

    • Show Columns

    • SHOW COLUMNS FROM table_name;

    • Describe Table Structure

    • DESCRIBE table_name;

    • Indexing

    • Create an Index

    • CREATE INDEX index_name
      ON table_name (column_name);

    • Drop an Index

    • DROP INDEX index_name
      ON table_name;

    • Transactions

    • Start a Transaction

    • START TRANSACTION;

    • Commit a Transaction

    • COMMIT;

    • Rollback a Transaction

    • ROLLBACK;

    • Backup and Restore

    • Backup a Database (Using 'mysqldump' command-line tool)

    • mysqldump -u username -p db_name > backup_file.sql

      • TIP: In order to run mysqldump , open your terminal, type mysqldump and you will see a list of
      usage examples. The -u flag stands for user and -p for password. In our case the user is root
      and after executing the command, you will be asked to type the password for that user.

    • Restore a Database

    • mysql -u username -p db_name < backup_file.sql

      • TIP: In order to run mysqldump , open your terminal, type mysqldump and you will see a list of
      usage examples. The -u flag stands for user and -p for password. In our case the user is root
      and after executing the command, you will be asked to type the password for that user.

    • Joins and Relationships

    • Inner Join

    • SELECT columns
      FROM table1
      INNER JOIN table2 ON table1.column = table2.column;

    • Left Join

    • SELECT columns
      FROM table1
      LEFT JOIN table2 ON table1.column = table2.column;

    • Right Join

    • SELECT columns
      FROM table1
      RIGHT JOIN table2 ON table1.column = table2.column;

    • Full Outer Join (Not natively supported in MySQL;
      can be simulated with UNION)

    • SELECT columns
      FROM table1
      LEFT JOIN table2 ON table1.column = table2.column
      UNION
      SELECT columns
      FROM table1
      RIGHT JOIN table2 ON table1.column = table2.column;
