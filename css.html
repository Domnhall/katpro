Welcome to CSS learn page, this site will help you to understand CSS and the syntax that you will use to write CSS. At the end of this lesson, you will be able to style a website by adding a background image,colors, and other decoration that you will learn to decorate your website.

CSS in full: Cascading Style Sheet

  • Basic CSS rule structure:

      selector {
      property: value;
      }

    • CSS syntax:

    • selector → h1- h1 is an example of a selector
    • declaration → {color:red; font-size:10px; }
    • property → color & font-size
    • value → red & 10px

    • CSS selectors:

    • css element selector
      p {
      color: red;
      }

    • css id selector
      #para {
      color: red;
      }

    • css class selector
      .title {
      color: green;
      }

    • css universal selector
      *body {
      font-family: monospace;
      }

    • how to add css:

    • external css → <link rel="stylesheet" href="name of your css file" >

    • internal css:
      <head>
      <style>
      body{
      background-color: #333;
      }
      .container{
      display: flex;
      }
      </style>
      </head>

    • inline css → <h1 style="color:red;" > paragraph </h1>

    • CSS comments:

    • /*comment here*/

    • CSS background:

    • CSS background color:

      div{
      background-color: blue;
      opacity: 0.3; /*this will give the opacity of 30%*/
      }

    • CSS background image:

      body{
      background-image: url(image.jpg); /*can be jpg,png,etc*/
      background-repeat: none; /*this depends on what you want not always none*/
      background-attachment: fixed; /*also depends if fixed or scroll*/
      }

    • CSS margins:

      css has properties for specifying margin of each side of an element such as:


    • margin-left, margin-right, margin-top, margin-bottom

      • all the margin property have the following values:

    • auto → for this the browser calculates the margin.
    • lenght → this specifies margin in px, pt, cm, etc.
    • % → specifies a margin in % of the width of the containing element.
    • inherit → specifies that the margin should be inherited from the parent element.

      • Tip:negative values can be used.

    • CSS padding

      CSS has a property for specifying padding of each side of an element:

    • → padding-right, padding-left, padding-top, padding-bottom.

      • All the padding property have the following values:

    • length → this specifies padding in px, pt, cm, etc.
    • % → specifies padding in % of the width of the containing element.
    • inherit → specifies that the padding should be inherited from the parent element.

      • example:

    • div {
      margin-top: 10px;
      margin-bottom: 5px;
      margin-left: 15px;
      margin-right: 20px;
      }

    • CSS Height/Width:

      The height and width properties may have the following values:

    • auto →The browser calculates the height and width
    • initial → Sets the height/width to its default value
    • % →Defines the height/width in percent of the containing block.
    • inherit →The height/width will be inherited from its parent value
    • length →Defines the height/width in px, cm, etc.

      • for example:

    • div{
      width: 200px;
      height: 90vh;
      background-color: blue;
      }

    • CSS Rounded Corners:

    • One value - border-radius: 15px;→the value applies to all four corners, which are rounded equally.

    • Two values - border-radius: 15px 50px;→first value applies to top-left and bottom-right corners, and the
      second value applies to top-right and bottom-left corners.

    • Three values - border-radius: 15px 50px 30px;→first value applies to top-left corner, second value applies
      to top-right and bottom-left corners, and third value applies to bottom-right corner

    • Four values - border-radius: 15px 50px 30px 5px;→first value applies to top-left corner, second value
      applies to top-right corner, third value applies to bottom-right corner, and fourth value applies to bottom-left corner

    • CSS grid:

      some examples

    • .grid-container{

      display: grid;
      column-gap: 20px;
      row-gap: 10px;

      /*short hand*/

      gap: 20px 10px;

      }

      • grid-template-columns property defines the number of columns
      in your grid layout, and it can define the width of each column.

      grid-template-rows property defines the height of each row.

    • CSS Animation:

      /*the animation code*/

    • @keyframes example{
      from{background-color: black;}
      to{background-color: white;}
      }

      • /*the element to apply the animation to*/

    • div{
      width: 50px;
      height: 50px;
      background-color: red;
      animation-name: example;
      animation-duration: 5s;
      }

      • /*multiple changes*/

    • @keyframes example{
      0% {background-color: pupple;}
      25% {background-color: black;}
      50% {background-color: blue;}
      100% {background-color: red;}
      }

      • /*the element to apply the animation to*/

    • div{
      width: 50px;
      height: 50px;
      background-color: red;
      animation-name: example;
      animation-duration: 5s;
      }

      • Tip:you can also use this properties below.

    • →animation-delay
    • →animation-iteration-count
    • →animation-direction
    • →animation-timing-function
    • →animation-fill-mode

    • CSS gradients:

      /*css linear gradient*/

    • background-image: linear-gradient(direction, color1, color2, ...);

      • example:

    • #grad{
      background-image: linear-gradient(right, red, yellow);
      }

      • /*css Radial gradient*/

    • background-image: radial-gradient(shape size at position, start-color, ..., last-color);

      • example:

    • #grad{
      background-image: radial-gradient(red 15%, yellow 5%, green 40%);
      }

      • /*css conic gradient*/

    • background-image: conic-gradient([from angle] [at position,] color [degree], color [degree], ...);

      • example:

    • #grad{
      background-image: conic-gradient(red, yellow, green);
      }

    • CSS units

      absolute units:

    • cm→centimeters
    • mm→millimeters
    • px→pixels (1px = 1/96th of 1in)
    • in→inches (1in = 96px = 2.54cm)
    • pc→picas (1pc = 12 pt)
    • pt→points (1pt = 1/72 of 1in)

      • relative units:

    • em→Relative to the font-size
      of the element (2em means 2
      times the size of the current font)
    • vh→Relative to 1% of the height of
      the viewport*
    • vmax→Relative to 1% of viewport's*
      larger dimension
    • vw→Relative to 1% of the width of the
      viewport*
    • rem→Relative to font-size of the root
      element
    • %→Relative to the parent element
    • ex→Relative to the x-height of the
      current font (rarely used)
    • ch→Relative to width of the "0" (zero)
